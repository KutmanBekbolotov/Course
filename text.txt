# Решение уравнения теплопроводности методом Дюамеля

Этот репозиторий содержит программу на языке Python для численного решения уравнения теплопроводности с использованием метода Дюамеля. Программа предназначена для моделирования распределения температуры в одномерном стержне в зависимости от времени.

## Установка

Для запуска программы вам понадобится Python версии 3.x, а также библиотеки NumPy и Matplotlib. Установите их, выполнив следующие команды:

```bash
pip install numpy
pip install matplotlib

Использование
Запустите файл main.py с помощью интерпретатора Python.
Программа автоматически рассчитает и визуализирует распределение температуры в стержне для заданного временного интервала.

Параметры программы
В программе можно настроить следующие параметры:

Длина стержня L.
Время моделирования T.
Коэффициент теплопроводности alpha.
Количество узлов по координате Nx.
Количество временных шагов Nt.
Измените значения этих параметров непосредственно в коде программы, чтобы настроить моделирование по вашим требованиям.
